# 🔨 ForgeRL

An OpenEnv-compatible RL environment where LLMs learn to orchestrate multi-agent teams to build software autonomously.

**Meta PyTorch × HuggingFace OpenEnv National Hackathon Submission**

## Quick Start

1. Click the **Reset** button to start a new episode
2. Select an action from the dropdown
3. Click **Step** to execute and see the reward
4. Watch as the meta-agent orchestrates sub-agents through the full SDLC!

## Themes Covered

- 🤝 Multi-Agent Interactions (9 sub-agents)
- 🧠 Long-Horizon Reasoning (50-300+ steps)
- 🔧 Professional Tasks (real tools)
- 📈 Self-Improvement (adaptive curriculum)
- 👁️ Fleet AI Oversight
- 👨‍💼 Expert-in-the-Loop
- 💰 Token-Scaled Rewards
